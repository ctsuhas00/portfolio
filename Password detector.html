Featured Project

Password Detection System

A Verilog HDL design that uses a Finite State Machine to recognize a predefined input sequence and grant access.

Completed Duration: 2 weeks
Verilog HDL FSM Xilinx Vivado ModelSim

Project Overview

A digital logic design project exploring secure sequence detection using Finite State Machines, implemented and verified entirely in simulation.

Problem Statement

Secure access control systems require reliable methods to verify user credentials and accurately detect predefined password sequences while rejecting unauthorized input.

Objectives

  • Design an FSM capable of recognizing a fixed input sequence
  • Implement the design in Verilog HDL
  • Verify correct behavior through simulation
  • Generate a clear access-granted output signal on match

Working Principle

The FSM transitions between states as each new input bit arrives, moving closer to an 'accept' state only when the sequence matches the expected password bit by bit. Any incorrect input resets the FSM to its initial state, and a correct full sequence asserts the access-granted output.

System Architecture

The design consists of a state register, next-state combinational logic describing valid transitions for each expected password bit, and an output stage that asserts the granted signal only in the accepting state.

Hardware & Software

Hardware Used

Software Used
