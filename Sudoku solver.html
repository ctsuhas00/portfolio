Featured Project

Sudoku Solver

A Python application that solves any valid Sudoku grid using a systematic backtracking algorithm.

Completed Duration: 1 week
Python Backtracking Data Structures

Project Overview

A software project focused on algorithmic problem-solving — automatically completing Sudoku puzzles of varying difficulty through recursive logic.

Problem Statement

Solving complex Sudoku puzzles manually is time-consuming and requires extensive trial-and-error reasoning.

Objectives

  • Represent the Sudoku grid as an efficient data structure
  • Implement a backtracking algorithm to test valid placements
  • Validate row, column, and box constraints at every step
  • Solve puzzles of varying difficulty reliably

Working Principle

The solver scans the grid for the next empty cell, tries each candidate digit 1–9, and recursively attempts to solve the rest of the grid. If a placement leads to a dead end, it backtracks and tries the next candidate, continuing until the grid is complete or exhausted.

System Architecture

The grid is represented as a 2D array, with helper functions to check row, column, and 3x3 box validity. A recursive solve() function drives the backtracking search until a full solution is found.
