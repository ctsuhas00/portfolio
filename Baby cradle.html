Featured Project

Automated Baby Cradle

An ESP32-powered cradle that listens for a baby's cry and rocks automatically, reducing the need for constant manual attention.

Completed Duration: 2 weeks
ESP32 Sound Sensor Servo Motor Wi-Fi

Project Overview

A smart-home style project applying embedded automation to infant care, designed to gently respond to a crying baby without requiring a parent to intervene immediately.

Problem Statement

Parents often struggle to continuously monitor and soothe infants, especially during sleep hours, and manual rocking can be time-consuming.

Objectives

  • Detect a baby's cry accurately using a sound sensor
  • Trigger an automatic rocking response
  • Minimize false triggers from background noise
  • Reduce the manual effort required from parents

Working Principle

A sound sensor continuously listens for audio patterns consistent with crying. When the threshold is crossed, the ESP32 activates a servo motor connected to the cradle's rocking mechanism, which runs for a set duration before checking again.

System Architecture

The ESP32 samples the sound sensor's analog output, applies a simple amplitude/duration filter to distinguish crying from ambient noise, and drives the servo motor through a PWM signal.

Hardware & Software

Hardware Used

ESP32 Sound Sensor / Microphone Module Servo Motor Power Supply Module

Software Used

Arduino IDE Embedded C
