Problem Statement

Secure access control systems require reliable methods to verify user credentials before granting access. Traditional systems must accurately detect predefined password sequences while preventing unauthorized access.

Objective

To design and implement a digital password detection system using Verilog and Finite State Machine (FSM) concepts for sequence recognition and access control.

Technologies Used

  • Verilog HDL
  • Finite State Machine (FSM)
  • Digital Logic Design
  • ModelSim / Simulation Tools
  • Xilinx Vivado

Description

Developed a password detection system using Verilog HDL based on Finite State Machine (FSM) architecture. The system continuously monitors input sequences and transitions between states to detect a predefined password pattern. Upon successful detection, the system generates an output signal indicating authorized access. The project demonstrates digital system design, state machine implementation, hardware description language programming, and simulation-based verification.

Key Features

  • FSM-Based Password Detection
  • Sequence Recognition Logic
  • Real-Time Input Monitoring
  • Authorized Access Indication
  • Verilog HDL Implementation
  • Simulation and Verification

Outcome

Successfully designed and simulated a digital password detection system capable of recognizing predefined input sequences, demonstrating proficiency in Verilog, FSM design, digital electronics, and hardware system development.
