Projects

Developed an ESP32-based smart baby cradle that detects infant cries using a sound sensor and automatically rocks the cradle using a motor mechanism. The system enhances infant comfort and reduces caregiver effort by providing immediate soothing action. It integrates IoT-enabled control, real-time monitoring, and automated response for improved baby care.
Developed a smart wearable system designed to detect user drowsiness and fatigue using sensors and real-time monitoring techniques. The device analyzes indicators of sleepiness and provides instant alerts through vibrations or sound notifications to improve safety and awareness. This system can help prevent accidents and enhance user well-being during daily activities.
Developed an Arduino-based autonomous robot capable of navigating its environment using IR sensors. The robot detects obstacles and changes its direction automatically without human intervention. The project involved sensor integration, motor control, and embedded programming, demonstrating concepts of automation, robotics, and real-time decision-making in autonomous systems.
Developed a Python-based Sudoku solver using the backtracking algorithm to efficiently solve Sudoku puzzles. The program systematically fills empty cells, validates each move according to Sudoku rules, and backtracks when conflicts occur. This project enhanced my understanding of recursion, algorithm design, problem-solving techniques, and optimization in software development.
Developed an FSM-based password detection system using Verilog HDL to verify a predefined sequence of inputs. The design utilized Finite State Machine (FSM) concepts to transition between states based on user inputs and generate authentication outputs. This project strengthened my knowledge of digital system design, sequential logic, state machine implementation, and hardware description languages.
